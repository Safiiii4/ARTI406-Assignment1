Data Cleaning


A review of the data was done to check for any missing value or any inconsistency in the dataset using the commands df.info() and df.isnull().sum() respectively. Missing values were seen in the "Sleep Disorder" column (219 cases).


To solve this problem, the missing values were substituted with "None". This is important to make sure there are no problems when doing any further calculations on the data.


There was no problem concerning duplicates or wrong data format. The data was organized in a good way from the beginning.


All missing values were solved in this way, since there were no more missing values observed when running df.isnull().sum() command.


This step improved the quality of our analysis significantly.


  

  



Visualizations and Findings:


Some visualizations were made in order to investigate relationships between variables of the dataset. These were histograms, scatter plots, box plots, and bar charts.


First, histograms were employed to see the distributions of sleep duration and sleep quality. Next, scatter plots were used to identify relationships between, for example, stress level and sleep quality or physical activity and sleep duration. Finally, box plots could be useful in comparing distributions by such factors as gender and BMI. At last, bar charts could help to compare factors such as occupation.


The selected methods seemed the most suitable ones in order to demonstrate relationships in the data.


The analysis led to the discovery of several key points. Namely, the average sleep duration is between six and eight hours. Also, high stress levels result in poor sleep quality and low sleep duration. On the other hand, there is a positive correlation between physical activity and sleep duration. Finally, one can note some dissimilarities between sleep durations among different occupations, BMIs, and ages.


As one can see, all of these findings show the effect of lifestyles on sleep health.
