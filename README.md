## Dataset Description

### Dataset Source
The dataset used in this project is the Sleep Health and Lifestyle Dataset, which contains information related to individuals' sleep patterns and daily habits.

### Dataset Size
The dataset consists of approximately:
- Rows: 374 records
- Columns: 13 features

### Features Description
The dataset includes the following key features:
- Gender: Gender of the individual
- Age: Age in years
- Occupation: Job type
- Sleep Duration: Average hours of sleep per night
- Quality of Sleep: Sleep quality rating
- Physical Activity Level: Level of daily physical activity
- Stress Level: Stress rating
- BMI Category: Body Mass Index classification
- Blood Pressure: Blood pressure measurement
- Heart Rate: Heart rate value
- Daily Steps: Number of steps taken per day
- Sleep Disorder: Presence of sleep disorders (if any)

### Purpose of the Dataset
The purpose of using this dataset is to perform Exploratory Data Analysis (EDA) to understand patterns and relationships between lifestyle factors (such as stress, activity, and BMI) and sleep quality. This helps in identifying key factors that influence overall sleep health.
